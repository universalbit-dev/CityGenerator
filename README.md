### Installation: [NodeJsv20.13.0 LTS](https://nodejs.org/en/blog/release/v20.13.0)

* Clone the project
```bash
git clone https://github.com/universalbit-dev/CityGenerator.git
cd CityGenerator
```
* Install packages with npm 
```bash
npm i && npm audit fix
```
